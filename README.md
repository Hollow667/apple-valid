# apple-valid

## how to use?
Node & npm required

### 1. clone this project
```
git clone https://github.com/kirahxr/apple-valid.git
```
### 2. instal dependencies
#### using npm
```
npm install 
```
#### using yarn
```
yarn
```
### 3. run this project
```
node index.js
```
